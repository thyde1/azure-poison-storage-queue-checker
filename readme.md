### Usage
Create user secrets in the format

```json
{
  "StorageAccounts": [
    {
      "Name": "{{ACCOUNT DISPLAY NAME}}",
      "ConnectionString": "{{STORAGE ACCOUNT CONNECTION STRING}}"
    },
    {
      "Name": "{{ACCOUNT DISPLAY NAME}}",
      "ConnectionString": "{{STORAGE ACCOUNT CONNECTION STRING}}"
    }
  ]
}
```
Run!
